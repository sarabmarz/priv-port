Run Santa Run!
2024
Unity / OpenCV
This project is an endless runner game that combines real-time hand gesture control using OpenCV with Unity-based game development. Built for the Image Processing and Vision course, it demonstrates advanced image processing techniques for hand tracking and gesture recognition.
  • Design elements are either handmade or from unity store
  • The fonts used in the game are Barriecito & Mountains of Chirstmas
  • Music and sound effects were sourced from Pixabay
